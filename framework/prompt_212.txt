### 翻译规则（大模型执行时严格遵守）

#### ⚠️ 核心原则：禁止对照表翻译
- 禁止使用任何形式的术语对照表、映射表、词对词替换、硬编码替换。

#### 必须原样保留的内容（不翻译）
- 所有代码、命令、文件路径、文件名、环境变量、命令输出

- 所有图片语法（Markdown `![alt](url)` 保持原样，但**后续要将 url 替换为本地路径**）

## Standard Tree JS

To customize the Tree View you must have a `{doctype}_tree.js` file in the doctype directory.
Below are all the options that can be customized.

For instance, if you want to configure the Account DocType, you'll have to create a file `account_tree.js` with the following contents.

`frappe.treeview_settings[&quot;Account&quot;] = {
	breadcrumb: &quot;Accounting&quot;,
	title: &quot;Chart of Accounts&quot;,

	filters: [
		{
			fieldname: &quot;company&quot;,
			fieldtype: &quot;Select&quot;,
			label: &quot;Company&quot;,
			options: &quot;Company 1\nCompany 2&quot;,
			on_change: handle_company_change(),
		},
	],

	get_tree_nodes: &quot;path.to.whitelisted_method.get_children&quot;,
	add_tree_node: &quot;path.to.whitelisted_method.handle_add_account&quot;,

	fields: [
		{
			fieldtype: &quot;Data&quot;,
			fieldname: &quot;account_name&quot;,
			label: &quot;New Account Name&quot;,
			reqd: true,
		},
		{
			fieldtype: &quot;Link&quot;,
			fieldname: &quot;account_currency&quot;,
			label: &quot;Currency&quot;,
			options: &quot;Currency&quot;,
		},
		{
			fieldtype: &quot;Check&quot;,
			fieldname: &quot;is_group&quot;,
			label: &quot;Is Group&quot;,
		},
	],

	ignore_fields: [&quot;parent_account&quot;],

	menu_items: [
		{
			label: &quot;New Company&quot;,
			action: function () {
				frappe.new_doc(&quot;Company&quot;, true);
			},
			condition: &quot;frappe.boot.user.can_create.indexOf('Company') !== -1&quot;,
		},
	],

	onload: function (treeview) {},

	post_render: function (treeview) {},

	onrender: function (node) {},

	on_get_node: function (nodes) {},

	extend_toolbar: true,

		{
