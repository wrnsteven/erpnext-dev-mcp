            You can create tabulated reports using server side scripts by creating a new Report.

Note: You will need Administrator Permissions for this.

Since these reports give you unrestricted access via Python scripts, they can only be created by Administrators. The script part of the report becomes a part of the repository of the application. If you have not created an app, read this.

Note: You must be in Developer Mode to do this

### 1. Create a new Report

![Script Report](/files/script-report.png)

Set Report Type as &quot;Script Report&quot;

- Set &quot;Is Standard&quot; as &quot;Yes&quot;

- Select the Module in which you want to add this report

- In the module folder (for example if it is Accounts in ERPnext the folder will be `erpnext/accounts/report/[report-name]`) you will see that templates for the report files will be created.

- In the `.js` file, you can set filters for the reports

- In the `.py` file, you can write the script that will generate the report

### 2. Add Filters

You can add filters in the `.js`. See an example below:

`
 frappe.queryreports[&quot;Accounts Receivable&quot;] = {
 &quot;filters&quot;: [
 {
 &quot;fieldname&quot;:&quot;company&quot;,
 &quot;label&quot;: (&quot;Company&quot;),
 &quot;fieldtype&quot;: &quot;Link&quot;,
 &quot;options&quot;: &quot;Company&quot;,
 &quot;default&quot;: frappe.defaults.getuserdefault(&quot;company&quot;)
 },
 ]
 }

`

- These properties are the same as you would set in a DocField in a DocType

### 3. Add the Script

In the `.py` file you can add the script for generating the report.

- In the `execute` method, two lists `columns` and `data` are returned

- Columns must be a list of dictionaries containing fields like fieldname, label, fieldtype, options,width. For example:

`columns = [{
 &quot;fieldname&quot;: &quot;account&quot;,
 &quot;label&quot;: _(&quot;Account&quot;),
 &quot;fieldtype&quot;: &quot;Link&quot;,
 &quot;options&quot;: &quot;Account&quot;,
 &quot;width&quot;: 300
},
{
 &quot;fieldname&quot;: &quot;currency&quot;,
 &quot;label&quot;: _(&quot;Currency&quot;),
 &quot;fieldtype&quot;: &quot;Link&quot;,
 &quot;options&quot;: &quot;Currency&quot;,
}]
`

- You can use all server side modules to build your report.

- For example see existing reports. (Balance Sheet)

### 4. Add link for your report on the module page

![Module Page](/files/script-report-1.png)

- In the module folder (for example if it is Accounts in ERPNext the folder will be `erpnext/config/accounts.py`) you will see labels and items for various sections. The new report can be added in the item list as show in the example:

`def get_data():
 return [{
 &quot;label&quot;: _(&quot;Accounting Statements&quot;),
 &quot;items&quot;: [{
 &quot;type&quot;: &quot;report&quot;,
 &quot;name&quot;: &quot;Balance Sheet&quot;,
 &quot;doctype&quot;: &quot;GL Entry&quot;,
 &quot;is_query_report&quot;: True
 }]
 }]
`
