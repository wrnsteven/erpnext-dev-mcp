系统用户和网站用户的区别

问题：我已经将我的员工添加为用户并分配了角色，但他们仍然无法在登录时查看仪表板。

答案：

ERPNext 中有两种用户类型。

  • 系统用户：他们是典型的内部用户，如员工、经理和管理员，需要广泛访问 ERPNext 系统以执行各种业务操作。他们拥有更高级别的访问权限和更详细的权限，允许他们与多个模块交互、生成报告和自定义工作流程。
  • 网站用户：他们是典型的外部用户，如客户或供应商，通过网络门户与 ERPNext 系统交互。他们的访问权限仅限于与其需求相关的功能，例如下订单、查看订单状态或管理其账户信息。与系统用户使用的全面后端界面相比，他们的界面更简单。

以下是 ERPNext 中系统用户和网站用户的表格对比

系统用户 网站用户
此类型有助于管理内部活动和操作 有助于管理外部操作和交易
提供对 ERPNext 模块和功能的完全访问 提供对 ERPNext 模块和功能的受限访问
此类别的典型用户是内部员工 此类别的用户角色包括外部各方，如客户、客户等
由 ERPNext 中的管理员创建 可以自行注册或由管理员创建
提供对公司范围内数据和报告的访问 仅提供对个人数据和有限报告的访问
可以查看、创建、编辑和删除记录 主要可以查看和创建与其相关的记录
在所有模块中具有完整的数据输入能力 仅具有与其角色相关的数据输入能力
