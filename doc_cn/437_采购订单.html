# 采购订单

采购订单是您与供应商之间具有约束力的合同，承诺在特定条件下购买一组物品。

它类似于销售订单，但不同的是，您不会将其发送给外部方，而是保留用于内部记录。

!图片

前提条件

在创建和使用采购订单之前，建议您首先创建以下内容：

- 供应商 - 物料

如何创建采购订单

采购订单可以从物料请求或供应商报价单自动创建。

- 进入采购订单列表，点击"新建"。 - 选择供应商和需求日期。 - 在物料表格中，通过代码选择物料，您可以更改每个物料的需求日期。 - 设置数量，如果物料主数据中设置了价格，则价格会自动获取。 - 设置税费。 - 保存并提交。

!图片

2.1 设置仓库

    \0

- 在采购订单中选择一个供应商。 - 在物料默认下的物料表单中设置默认供应商。 - 存在一个类型为"采购"的物料请求。 - 点击供应商名称下方的"从未结物料请求获取物料"按钮。此时将出现一个对话框，其中包含默认供应商与采购订单中所选供应商相同的物料请求。选择物料请求并点击"获取物料"后，物料将从物料请求中获取。

> 注意：只要物料表格为空，"从未结物料请求获取物料"按钮就会显示。

!图片

功能

3.1 地址和联系人

- 选择供应商地址：供应商的账单地址。 - 选择发货地址：供应商的发货地址，他们将从这里发送物料。

地址、发货地址、联系人、联系人邮箱如果已在供应商主数据中保存，将会自动获取。

对于印度：

- 供应商和公司 GSTIN：您的供应商和您公司的商品及服务税识别号。 - 供应地点：对于 GST，供应地点是必需的。它由邦名和编号组成。

3.2 货币和价格表

您可以设置存储采购订单的货币。如果您设置了价格表，物料价格将从该列表中获取。勾选"忽略价格规则"将忽略"账务 > 价格规则"中设置的价格规则。

了解更多关于价格表和多币种交易的信息。

3.3 外协或"供应原材料"

设置"供应原材料"选项适用于外协场景，在这种模式下您提供用于制造产品的原材料。了解更多，请访问外协页面。

3.4 物料表格

    \0

- 您可以通过点击"更新物料"按钮在采购订单中添加、更新、删除物料。但是，您无法删除已收货的物料。 - 状态：提交后，您可以持有或关闭采购订单。

创建：从已提交的采购订单中，您可以创建以下内容： - 采购收货 - 表示您已收到物料的收货单。 - 采购发票 - 采购订单的发票/账单。 - 付款凭证 - 表示已针对采购订单付款的付款凭证。 - 日记账分录 - 记录在总账中的日记账分录。
