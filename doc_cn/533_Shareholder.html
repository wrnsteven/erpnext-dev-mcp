股东是指拥有公司股份的人。

股东是指持有公司至少一股股份的任何个人、公司或其他机构。由于股东是公司的所有者，他们以股票增值的形式获得公司成功带来的收益。如果公司业绩不佳，股东可能会因公司股价下跌而遭受损失。

公司的任何盈利或亏损都属于股东，因此股东是公司的负债方。

ERPNext 允许您跟踪所有股东信息，并维护股权转让、股东分户账和股份余额。股东通过股东 ID 进行唯一标识。通常该 ID 是一个以 'ACC-SH-' 开头的命名序列。此外，只要股东进行过一次交易，系统就会为其分配一个股权登记号码。这也是该股东的独特标识。

要访问股东列表，请前往：
