# 个人数据下载工具

个人数据下载工具使用户能够自动下载在使用 ERPNext 期间生成的所有个人数据。这包括您用户账户中的个人身份数据，如：用户名、全名、出生日期、电话号码、手机号码、位置、兴趣、个人简介、电子邮件签名、电子邮件、联系人、地址、通信记录等。还包括来自潜在客户和商机的数据，您保存的详细信息（如电话号码、手机号码、传真、网站和姓名）。

如何请求用户数据下载

要开始下载数据，用户必须登录，然后在 URL 字段中访问 [host-name]/request-data（例如 erpnext.com/request-data）。
