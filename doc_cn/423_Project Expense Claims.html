员工从事 Project 项目工作会产生各种费用，有时需要自己垫付，然后可以通过 Expense Claim 报销这些费用。

填写报销单时，员工可以指定产生该费用的 Project 项目。

!图片

根据为特定 Project 项目提交的费用报销，系统会在 Project 主数据的项目成本部分更新该项目的费用报销总金额。
