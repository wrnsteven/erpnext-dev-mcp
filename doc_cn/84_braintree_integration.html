要设置 Braintree，请前往 Explore > Integrations > Braintree Settings

设置 Braintree

要在你的 ERPNext 账户中启用 Braintree，你需要配置以下参数：

  • Merchant ID
  • Public Key
  • Private Key

如果需要，你可以设置多个 Braintree 支付网关。支付网关账户的选择将决定使用哪个 Braintree 账户进行支付。

启用服务后，系统将创建 Payment Gateway 记录和科目表中的账户类型为 Bank 的账户。

它还会创建支付网关账户。如果需要，你可以更改默认的银行账户并创建付款请求模板。

配置支付网关账户后，你的系统能够通过 Braintree 接受在线付款。
