电子商务设置

旧的购物车设置和产品设置已合并为一个单一的电子商务设置页面。

ERPNext 中的电子商务设置可被用户利用来管理和简化在线销售流程。这些设置使用户能够有效地管理产品列表、客户管理以及与其他 ERPNext 模块的无缝集成。

在 awesomebar（搜索栏）中输入E Commerce Settings / Webshop Settings

E Commerce Settings

每页产品数

定义在所有产品页面（/all-products）的产品列表中每页显示的产品数量。

筛选器和类别

筛选器和类别

你还可以向列表添加筛选器。有两种类型的筛选器：

  1. 字段筛选器：启用字段筛选器复选框，并添加你希望作为筛选依据的字段。这些字段也将用作按类别购物页面的类别。
  2. 属性筛选器：启用属性筛选器复选框，并添加你希望作为筛选依据的属性。

显示设置

  1. 隐藏变体：仅在产品列表中显示 Website Item Templates。
  2. 启用变体选择：如果 Item 有变体，则显示配置按钮。它可用于根据属性缩小到特定 item。
  3. 显示价格：在产品页面显示 Item 价格。
  4. 显示库存可用性：显示 Item 是否有库存。
  5. 显示库存数量：在产品页面显示可用库存。
  6. 允许将缺货产品添加到购物车：允许将缺货产品添加到购物车。
  7. 显示应用优惠券代码：结账时应用优惠券代码的附加功能。
  8. 显示联系我们按钮：显示联系我们按钮，客户可以用它来询问 Item。它将在系统中创建一个 Lead。
  9. 显示公开附件：在结账后生成的销售订单中为客户显示公开附件。

仅当属性筛选器被禁用时，你才能隐藏变体。

购物车

  1. 公司：指定网店所属的公司。
  2. 价格列表：提及获取 Item 价格时要考虑的价格列表。
  3. 启用购物车：添加将产品添加到购物车的功能。
  4. 默认客户组：添加在将产品添加到购物车时自动创建客户的默认客户组。

结账设置

  1. 启用结账：如果结账被禁用，当你的客户将产品添加到购物车时，他们可以点击请求报价按钮来获取报价。系统会生成一个报价单。如果启用，点击购物车中的下单按钮会生成一个销售订单。
  2. 支付网关账户：如果启用了结账，你必须配置PayPal 集成或Razorpay 集成。
  3. 支付成功 URL：支付完成后将用户重定向到所选选项。
  4. 在报价单中显示价格：如果结账被禁用，你可以选择是否在报价单中的 Website Items 上显示价格。
  5. 将报价单保存为草稿：如果结账被禁用，点击请求报价按钮时，你可以选择自动生成的报价单是自动提交还是保持在草稿状态。

附加功能

  1. 启用愿望清单：添加将产品添加到愿望清单的功能
  2. 启用评论和评分：为客户添加对 Website Items 发表评论和评分的功能。
  3. 启用推荐：添加在 Website Item 旁显示手动选择的推荐产品的功能。

按类别购物

按类别购物页面是一个现成的页面，列出链接到 Website Items 的类别。这些类别可以在筛选器和类别部分的字段筛选器表中配置。

你还可以选择一个幻灯片作为页面的横幅。

访客显示设置

  1. 对访客隐藏价格：你可以对访客用户（未登录的用户）显示或隐藏 Website Items 的价格。
  2. 操作时重定向：如果访客用户尝试将 Website Item 添加到购物车或愿望清单，你可以选择一个 URL 来重定向他们。默认情况下，访客用户被重定向到/login页面。你可以通过在此添加 URL 来将它们重定向到你选择的自定义 Web 表单，例如/contact-us
