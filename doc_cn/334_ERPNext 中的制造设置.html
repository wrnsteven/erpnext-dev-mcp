# ERPNext 中的制造设置

制造设置用于配置生产工作流程、管理物料清单（BOM）、跟踪工单，以及监督库存，从而简化运营并确保高效的生产管理。

> 首页 > 制造 > 设置 > 制造设置

原材料消耗

!图片

1. 允许连续物料消耗

启用后，材料可以在单个工单中消耗而无需立即生产成品。当生产一个或多个耗时的产品时，此功能非常有用。

例如，单个产品需要一个月才能制造完成，而原材料每天都在消耗。在常规情况下，这对于库存单据来说是不可行的。启用此选项将允许您创建物料消耗的库存单，而无需创建倒冲单据。最终结果是您可以看到仓库中消耗的库存，并可以在稍后阶段更新成品的最终制造单据。

2. 基于以下条件倒冲原材料

此处选择的方法将用于倒冲原材料：1. 制造转移的物料 2. BOM

3. 根据 BOM 验证组件和数量

注意：此功能将从 V15 版本开始可用

!图片

如果"倒冲原材料基于"设置为 BOM，则用户可以根据 BOM 验证组件数量。为此，他们需要在"制造设置"中启用"根据 BOM 验证组件数量"复选框。

4. 从消耗单获取原材料成本

!图片

如果启用了"允许连续物料消耗"，则用户可以看到"从消耗单获取原材料成本"选项。

"从消耗单获取原材料成本"决定了生产中使用的原材料成本是否应自动从制造过程中记录的消耗单派生。启用后，ERPNext 将根据实际消耗的材料计算成本，确保准确的成本跟踪和财务报告。

产能规划

产能规划是组织根据资源和现有工单决定是否接受新订单的过程。

!图片

1. 禁用产能规划

如果勾选，则不会进行产能规划。启用它将有助于根据资源和现有工单决定是否接受新订单。

2. 允许加班

启用后将允许在工作站工作时间之外创建工单、工序卡等。

3. 允许节假日生产

启用后，即使是组织假期列表中标记为假日的日期，也允许进行生产活动。

4. 产能规划提前期（天）

此处指定的天数表示将提前多少天启动产能规划活动以进行生产。

5. 工序间隔时间（分钟）

这指定了两个工序之间应保持的时间跨度（以分钟为单位）。

生产默认仓库

!图片

1. 默认在制品仓库

此仓库将自动更新到工单的"在制品"仓库字段中。

2. 默认成品仓库

此仓库将自动更新到工单的"目标仓库"字段中。

3. 默认废料仓库

此仓库将自动更新到工单的"废料仓库"字段中。

销售订单和工单的超额生产

!图片

1. 销售订单超额生产百分比

它允许您指定生产可以超过销售订单数量的百分比。

2. 工单超额生产百分比

它定义了实际生产数量可以超过工单中指定计划数量的允许百分比。

工序卡

!图片

1. 将纠正操作成本计入成品估价

启用后，在计算成品估价时也将包含纠正操作类型的成本。

2. 允许超额物料转移

启用后，物料转移按钮将可见，并允许您在针对工序卡完成原材料需求后仍然转移原材料。

这在转移的原材料损坏且需要额外转移原材料以生产与预期相同数量的成品的情况下特别有用。

其他设置

!图片

1. 自动更新 BOM 成本

如果勾选，BOM 成本将根据原材料的估值单价/价格表单价/最近采购价自动更新。

2. 从子组件设置操作成本/废料项目

在工单中使用"多级 BOM"的情况下，如果用户希望在不使用工序卡的情况下也将子组件成本添加到成品项目以及废料项目中，则需要启用此选项。

3. 从工单创建序列号/批次

如果勾选，系统将在提交工单时自动为成品创建序列号/批次。

4. 允许编辑工单中的物料和数量

启用后，系统将允许用户编辑工单中的原材料及其数量。如果用户在工单物料表中更改了数量，系统不会根据 BOM 重置数量。
