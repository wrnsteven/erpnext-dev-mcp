# 个人数据删除工具

个人数据删除工具使用户能够删除其账户，并匿名化用户在使用 ERPNext 期间生成的所有个人可识别数据。也就是说，个人可识别信息将被随机化。这包括您用户账户中的个人可识别数据，如：用户名、全名、出生日期、电话号码、手机号码、位置、兴趣、个人简介、邮件签名、电子邮件、联系人、地址、通信记录等。它还包括来自线索和商机中的数据，您保存的详细信息，如电话号码、手机号码、传真、网站和名称。

但是，这不包括法律要求企业保留的数据。

如何申请账户删除

要开始删除用户账户和个人可识别数据，您需要访问 URL 字段中的 [host-name]/request-for-account-deletion（例如 example.erpnext.com/request-for-account-deletion）。

!图片

输入与您 ERPNext 账户关联的电子邮件。提交请求后，您将收到成功响应。

!图片

这将向与用户关联的电子邮件地址发送一封包含数据删除验证链接的电子邮件。
