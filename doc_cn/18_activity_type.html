活动类型

ERPNext 中的 Activity Type 帮助您维护可以制作工时表的不同类型的活动。

要访问 Activity Type，请前往：

Home > Projects > Time Tracking > Activity Type

如何创建活动类型

  1. 转到 Activity Type 列表并点击"新建"。
  2. 添加活动类型的主题。
  3. 如有需要，指定"默认成本费率"。
  4. 如适用，提及"默认计费费率"。
  5. 保存。

默认情况下，ERPNext 中会创建以下活动类型。

  • 规划
  • 研究
  • 提案撰写
  • 执行
  • 沟通

此外，也可以使用 Activity Cost 为个别员工配置默认成本费率和默认计费费率。
