# 用户 ID 是基于电子邮件地址创建的

当用户被重命名时，其值会在所有现有记录（包括变更日志）中更新。因此，这是一个资源密集型操作。如果在重命名用户时遇到超时问题，建议使用更新后的电子邮件地址创建新的用户 ID，然后禁用旧用户。

要更改用户的电子邮件 ID，请按照以下步骤操作：

> 用户列表 → 打开用户 → 点击菜单 → 重命名 → 输入新的电子邮件 ID。保存
