# 本地化覆盖

为了符合适用于POS软件的最新财务法规，ERPNext会自动在链式日志中注册所有销售和支付交易。

如果您的国家设置为"法国"，则不允许删除销售和支付交易，即使向用户授予了适当的权限。

请注意，ERPNext尚未完全符合2016年财务法规。

# 链式日志报告

提供了一个名为"交易日志报告"的专用报告，用于验证日志链是否未被破坏。

如果"Chain Integrity"列的状态为"True"，则表示链未被破坏。这意味着您可以查阅系统中发生的完整事件链。

如果"Chain Integrity"列的状态为"False"，则表示链中存在差异。在这种情况下，前一行已在数据库中被删除或篡改。这很可能是欺诈行为的结果。
