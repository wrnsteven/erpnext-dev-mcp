管理员

ERPNext 中的'Administrator'凌驾于 System Manager 之上，拥有 ERPNext 账户的所有权限。

System Manager 也拥有系统中大多数项目的权限，但 Administrator 拥有无限制的访问权限。管理员确保 ERPNext 系统的整体运行满足组织的需求。

  1. 如果您的 ERPNext 账户是由我们（Frappe Technologies Pvt. Ltd.）托管在云端，您可以通过 Frappe Cloud Dashboard 访问"Administrator"用户。
  2. 对于云托管账户，升级从后端管理。我们保留管理员登录凭证，以便能够从后端升级所有客户的 ERPNext 账户。
  3. 由于在一台服务器上我们可能托管许多客户的 ERPNext 账户，作为安全措施，我们无法与任何云托管用户共享管理员账户的凭据。（例外情况是，如果您购买了大量用户并且您的账户专门托管在一台服务器上）。
  4. 对于自托管的本地账户，管理员凭据由账户用户保管。
