Installation Note 是在产品/商品在客户处安装或设置时创建的。

例如，如果您销售空调，服务人员通常会在送货时陪同，将空调安装在客户处。服务人员可以创建一条"Installation Note"，以跟踪安装的日期和详情。

要访问 Installation Note 列表，请前往：

> Home > Stock > Serial No and Batch > Installation Note

1. 如何创建 Installation Note

Installation Note 通常从 Delivery Note 创建。您也可以手动创建，然后点击"From Delivery Note"按钮从选定的 Delivery Note 中获取详情。
