文档标题

你可以根据属性自定义文档标题，以便在列表视图中获取有意义的信息。

例如，报价单上的默认标题是客户名称，但如果你只与少数客户打交道，并向每个客户发送大量报价单，你可能需要自定义。

设置标题字段

从 ERPNext Version 6.0 开始，所有交易都有一个"Title（标题）"属性。如果没有标题属性，你可以添加一个自定义字段作为标题，并通过Customize Form（自定义表单）设置Title Field（标题字段）。

你可以使用 Python 风格的字符串格式化在Default（默认值）或Options（选项）中设置该属性的默认值

  1. 要编辑默认标题，请进入 Customize Form（自定义表单）
  2. 选择要更改其标题字段的表单。
  3. 在表单中编辑Title Field（标题字段）。

定义标题

你可以通过在花括号{}中设置文档属性来定义标题。例如，如果你的文档有字段customer_name，你可以将其指定为表单的标题。

或者，你也可以在 Customize Form（自定义表单）中将特定字段定义为"Title Field（标题字段）"。

固定或可编辑标题

如果你的标题生成为默认标题，用户可以通过点击文档标题来编辑它。

如果你想要固定标题，你可以在Options（选项）属性中设置规则。这样，每次更新文档时，标题都会自动更新。
