从 Excel 复制粘贴多条记录

如果你有一些保存在 Excel 表格中的记录序列，需要映射到 ERPNext 的子表中，可以使用此功能。

比如说，你在 Excel 表格中有一份物品列表，需要复制到销售订单的"物品"子表中。

从 Excel 复制粘贴记录的步骤

  • 在 Excel 或文本编辑器中准备源数据，每列用制表符分隔。

  • 拖动选择记录，点击复制菜单按钮或按 Ctrl + C（Cmd + C）

情况一：Excel 表格的第一列应该是列标题及其内容。

情况二：当没有定义的列标题时，数据将映射到可见列。

  • 将光标放置到子表的目标输入字段中，然后粘贴。与通过上传文件导入不同，这个复制粘贴功能会自动触发字段变更事件。

出于性能考虑，每次粘贴的记录不应超过 100 条。
