使用 Customize Form，我们可以为 ERPNext 中任何 Doctype 的任何字段设置默认值。

步骤：

1. 使用快速搜索栏或通过任意 DocType 的菜单选项打开 Customize Form。这将打开一个界面，您可以在其中查看构成此 DocType 的所有字段。

!图片

2. 展开需要设置默认值的字段，然后在 Display -> Default 下输入默认值。

!图片
