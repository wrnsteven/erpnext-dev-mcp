# 销售税和费用模板

销售税和费用可应用于您销售的任何商品。

从该表单创建的模板可用于销售订单和销售发票。

对于您想在税收模板中使用的税务账户，您必须将该特定账户的账户类型字段设置为"税务"。ERPNext 设置税款的方式是通过模板。其他可能适用于您发票的费用类型（如运费、保险等）也可以配置为税款。 了解如何设置税款，请访问此页面。

要访问销售税和费用模板，请前往：

> 首页 > 销售 > 设置 > 销售税和费用模板

了解如何设置税款，请访问此页面

2. 功能

2.1 销售税和费用表

考虑税费对象：合计 - 适用于所有商品的合计。valuation - 适用于每个商品。valuation 及合计 - 同时适用于两者。查看这篇文章了解其中的区别。

参考行号 #：如果税款基于"上一行合计"，您可以选择将哪一行作为此计算的基础（默认为上一行）。 !图片

此税款是否计入基本单价？：如果勾选，税款金额将被视为已包含在交易的商品表中显示的单价/打印金额中。当您想向客户提供含税价格时，这很有用。为了处理含税价格，系统通过先扣除要应用的税款金额来计算净额，然后再计算其中的税款。

账户科目： 税款将入账的账户分类账。如果您选择 VAT 或任何其他预设科目，税率将自动填充。

成本中心： 如果税款/费用是收入（如运费）或支出，则需要通过成本中心记账。

描述： 税款的描述（将打印在发票/报价单上）。

税率： 税款税率，例如：14 = 14% 税款。

金额： 要应用的税款金额，例如：100.00 = ₹100 税款。

您在模板中定义的税率是所有商品的的标准税率。如果某些商品应适用不同的税率，您可以通过为商品或商品组设置商品税模板来覆盖标准税率。
