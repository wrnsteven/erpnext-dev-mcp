支票打印模板

支票打印模板允许您定义银行支票的模板。

企业需要向供应商、员工等各方进行付款。付款方式有多种，如现金、NEFT 或支票。如果您通过支票付款，还可以基于付款凭证在 ERPNext 中创建用于打印支票的打印格式。

访问支票打印模板，请前往：> 首页 > 会计 > 支票打印模板

使用支票打印模板，您可以生成新的支票打印格式。它将基于银行提供的支票格式创建。

如何创建支票打印模板

  • 进入支票打印模板列表，点击"新建"。
  • 您可以设置支票中各个项目的位置。
  • 保存。

在支票打印模板中，对于每个值（如收款人、日期），会根据该值在支票上的打印位置提供精确的坐标。坐标以厘米为单位。以下是结构表示：

1.1 通过扫描创建新格式

为了加快创建新支票打印格式的速度，您可以上传支票的扫描图像。系统会根据支票的扫描图像自动更新各个值的坐标，如收款人名称、金额、日期、金额大写等。

1.2 手动创建新的打印格式

如果预览效果良好，点击"创建打印格式"按钮为支票创建新的打印格式。根据支票打印模板中提供的值，系统将自动生成支票打印格式的 HTML 脚本。

您可以基于支票上想要打印的位置手动提供每个值的坐标，并使用 HTML/CSS 进行自定义。

预览

根据为所有值提供的坐标，将显示预览，展示这些值在支票上的打印效果。

1.3 打印支票

为支票生成的新打印格式将在付款凭证表单中可见。创建付款凭证后，您将能够把交易详情打印到支票上。
