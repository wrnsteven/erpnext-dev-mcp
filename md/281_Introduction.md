## 什么是 ERPNext？

ERPNext 是一款业务流程管理软件，帮助公司全面掌控运营状况，包括库存跟踪、财务管理、项目处理和客户维护等各个方面。可以把它想象成您业务的神经系统，将所有事务整合到一个统一的平台中。

大多数 ERP 系统都笨重、昂贵，定制起来也很头疼。ERPNext 与众不同。它是开源的、云端就绪的，并且设计灵活。您可以拥有企业级系统的强大功能，同时避免通常的复杂性。

## 为什么选择 ERPNext？

经营企业已经够难了，您的 ERP 系统不应该让情况变得更复杂。

使用 ERPNext，您可以获得满足所有业务需求的一整套内置工具，具体包括：

- 完整的工具套件：从资产管理到财务会计到库存管理，您需要的一切都已内置其中。

- 无供应商锁定：开源意味着您拥有自己的数据，并可以自由地进行定制。

- 易于使用：简洁、现代的界面，不会让人感觉像是上世纪 90 年代的产品。

- 成本效益高：没有高昂的许可费用，只有功能强大且好用的软件。

无论您是小型企业还是成长中的企业，ERPNext 都能帮助您保持井然有序、自动化繁琐的事务，并专注于真正重要的事情——经营您的业务。

## 主要功能

ERPNext 将帮助您：

- 跟踪组织的财务指标。

- 跟踪所有发票和付款。

- 了解库存中可用产品的数量。

- 识别和跟踪您的关键绩效指标（KPI）。

- 识别并处理未解决的客户咨询。

- 管理员工工资。

- 分配任务并跟进。

- 维护所有客户、供应商和联系人的数据库。

- 准备报价单。

- 跟踪预算和支出。

- 根据实际原材料成本、机械设备成本和人工成本确定有效的销售价格。

- 获取维护计划的提醒。

- 发布您的网站。

以及更多功能。

## 技术架构

- [Frappe Framework](https://github.com/frappe/frappe)：一款使用 Python 和 JavaScript 编写的全栈 Web 应用框架。该框架为构建 Web 应用提供了强大的基础，包括数据库抽象层、用户认证和 REST API。

- [Frappe UI](https://github.com/frappe/frappe-ui)：一个基于 Vue 的 UI 组件库，提供现代化的用户界面。Frappe UI 库提供了丰富的组件，可用于在 Frappe Framework 之上构建单页应用。
