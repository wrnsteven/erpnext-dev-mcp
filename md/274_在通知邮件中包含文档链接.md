# 在通知邮件中包含文档链接

在 ERPNext 中发送邮件通知时，很多时候我们需要将文档 ID 作为链接的一部分包含在邮件内容中。例如，如果我们为每个新的物料请求设置了提醒（通知），那么我们也希望能将新物料请求的链接通过邮件发送给我们。要设置此功能，请按照以下步骤操作：

进入通知列表，点击“新建”。

为任何 DocType 创建新的通知。这里我们以在创建每个新物料请求时发送电子邮件通知为例，如下图所示。

![图片](../../images/274_6WPgqY6.png)

要将关联记录的文档 ID 作为链接发送，请将以下代码片段复制到邮件内容中：

`&lt;a href=&quot;{{frappe.utils.get_url_to_form(doc.doctype, doc.name)}}&quot;&gt;{{doc.name}}&lt;/a&gt;`

![图片](../../images/274_vHK6tDW.png)

现在，当创建新的物料请求时，由于上述设置的通知，将收到以下电子邮件。

![图片](../../images/274_3WOeTEv.png)

点击链接后，您将跳转到新的物料请求。

![图片](../../images/274_4hB36zh.png)
