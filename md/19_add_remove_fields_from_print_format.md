# 从打印格式添加/删除字段

为了解释，我们以采购订单为例，其打印格式中有以下字段，在这里，我想在供应商名称下添加"供应商联系人名称"，并从项目表中删除"每单位重量"列。

**步骤 1:** 导航到要更改打印格式的文档的打印视图 -> 菜单 (...) -> 自定义

**步骤 2:** 按名称搜索字段并将其添加到所需位置

*使用相同的拖放方法也可以删除字段。*

**步骤 3:** 要从文档的子表添加/删除字段：找到表格 -> 选择列 -> 根据需要勾选/取消勾选。
