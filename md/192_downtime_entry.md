# Downtime Entry

To access the Downtime Entry, go to:

> Home > Manufacturing > Production > Downtime Entry

![](/files/downtime-entry.png)

Downtime Entry document is used to capture the machine's Downtime in minutes. User needs to manually create the downtime record by selecting the proper machine
