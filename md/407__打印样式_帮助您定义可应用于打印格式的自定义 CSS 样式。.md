# "打印样式"帮助您定义可应用于打印格式的自定义 CSS 样式。

ERPNext 提供了预设的文档打印样式。您还可以使用 CSS 创建新样式，并将其应用于所有打印格式。

ERPNext 中的标准打印样式包括：Monochrome、Modern、Redesign 和 Classic。

要创建新的打印样式，请转到：

> 首页 > 设置 > 打印样式

## 如何创建新的打印样式？

- 进入打印样式列表，点击"新建"。

- 输入打印样式的名称。

- 输入将定义样式外观的 CSS。

- 保存。

您在此处创建的样式适用于标准和自定义打印格式。要了解各种可用的 CSS 类，您可以创建一个标准打印格式，在新页面中打开并查看源代码。

可以从[打印设置](/erpnext/print-settings)设置默认打印样式。

所有打印格式样式都基于 Bootstrap（3 版本）CSS 框架。

![图片](../../images/407_screenshot_2023-12-29_at_120810_pm.png)

如果您已启用开发者模式并勾选了"标准"，系统将为该打印样式生成 JSON 文件。您可以使用此方式贡献一个默认打印样式。
