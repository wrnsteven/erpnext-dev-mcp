**Project Update 是指项目状态的更新信息，可以发送给项目的所有相关方。**

每当您需要向相关方通报项目情况时，都可以向他们发送 Project Update。

![图片](../../images/428_image61c72.png)

要访问 Project Update，请前往：

> 首页 > 项目 > 项目 > Project Update

## 如何创建 Project Update

- 进入 Project Update 列表，点击"新建"。

- 添加您要发送更新的项目名称。

- 在子表中添加您想要接收 Project Update 的用户名。这些用户可以是您的客户，也可以是任何内部或外部的相关方。

- 保存并提交。

一旦您提交了 Project Update，一封包含项目更新内容的邮件就会发送给选定的相关方。

### 其他详情

展开包含用户名的表格后，您可以添加以下详情：

- 您可以选择在发送 Project Update 时是否向特定用户发送欢迎邮件。

- 您可以选择是否允许特定用户查看 Project Update 中的附件。

- 您可以在"项目状态"字段中添加项目的当前状态描述，该描述将展示项目的进展情况。
