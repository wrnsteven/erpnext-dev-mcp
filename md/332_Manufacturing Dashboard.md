## Number Cards

在 ERPNext 中，Number Cards 是一种小部件或可视化组件，用于在 Dashboard 中显示关键指标或数字数据。它们通常用于展示快速、高层次的洞察，帮助用户跟踪重要的业务绩效指标或运营统计数据。

![图片](../../images/332_332_CleanShot%202026-02-19%20at%2011.41.16@2x.png)

- **Monthly Total Work Order** :- 您将获得处于"打开"、"进行中"和"已完成"状态的 Work Orders 总数。显示在 Number Card 中的是当前日期前一个月内创建的 Work Orders。

- **Monthly Completed Work Order** :- 用户将获得处于"已完成"状态的 Work Orders 总数。显示在 Number Card 中的是当前日期前一个月内创建的 Work Orders。

- **Ongoing Job Card** :- 用户将获得未处于"已完成"状态的 Job Cards 总数。

- **Monthly Quality Inspection** :- 用户将获得处于"已提交"状态的 Quality Inspection 记录总数。显示在 Number Card 中的是当前日期前一个月内创建的 Work Orders。

## Dashboard

打开 Manufacturing 模块时，您可以在 Dashboard 中看到一些预定义的图形报表。这些报表是完全可自定义的，您可以选择显示或隐藏的内容，也可以配置报表所依据的指标。

![图片](../../images/332_332_CleanShot%202026-02-19%20at%2011.52.51@2x.png)

### Produced Quantity

该图表将提供有关上一年度按季度统计的总生产数量（使用 Work Order）的信息。用户也可以按日、周、月、年为单位查看图表数据。

### Completed Operation

该图表将提供有关上一年度按季度统计的已完成操作总数的息。用户也可以按日、周、月、年为单位查看图表数据。

![图片](../../images/332_332_image31eb3a.png)

### Work Order Analysis

该图表将提供有关基于"未开始"、"进行中"、"已停止"、"已完成"状态的 Work Orders 数量的信息。此 Donut 图表将基于上一年度的 Work Order 数据提供信息。您可以通过点击 **Filter** 按钮更改日期范围。

### Quality Inspection Analysis

该图表将提供有关基于"已接受"和"已拒绝"状态的 Quality Inspections 数量的信息。图表类型为 Donut，它将基于上一年度的 Quality Inspection 数据提供信息。您可以通过点击 **Filter** 按钮更改日期范围。

### Pending Work Orders

该图表将提供有关基于老化天数的待处理 Work Orders 数量的信息。老化天数是根据当前日期与计划开始日期/实际开始日期之间的差异计算的。

### Last Month Downtime Analysis

该图表将提供有关机器在上个月停止工作的总分钟数的信息。这有助于操作员了解哪台机器表现不佳，需要维护。

![图片](../../images/332_332_image60f6db.png)

### Work Order Quantity Analysis

该图表将提供有关上一年度每月基于 Work Orders 的待处理和已完成数量的信息。

### Job Card Analysis

该图表将提供有关上一年度每月处于"待处理"和"已完成"状态的 Job Cards 总数的信息。
