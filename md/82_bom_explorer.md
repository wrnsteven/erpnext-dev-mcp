# BOM Explorer

BOM Explorer 报表以树形视图的形式显示物料清单（BOM）及其原材料的完整层次结构。

你需要在筛选器中选择 BOM，然后报表将显示制造该项目的所需原材料数量和废料。这是所有装配、子装配（即所有展开项目）的完整树形视图。

使用 BOM Explorer，用户可以查看所有展开项目的原材料和废料数量并打印。
