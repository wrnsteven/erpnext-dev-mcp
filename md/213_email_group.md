# 邮件组

邮件组是订阅者电子邮件 ID 的列表，您可以通过它发送新闻通讯。

您可以通过将订阅者分组到不同的组来管理他们。

您可以根据客户的地区或他们感兴趣的产品，将现有客户和潜在客户分组。您可以将所有员工的电子邮件 ID 分组，以便与他们共享信息。

访问邮件组列表，请前往：

首页 > 设置 > 电子邮件 > 邮件组

## 1. 如何创建邮件组

进入邮件组列表，点击"新建"。

- 输入"标题"。

- 点击"保存"

- 点击"操作" > "添加订阅者"。

- 输入电子邮件地址。

- 点击"添加"

## 2. 功能

### 2.1 从交易中导入电子邮件 ID

您可以从现有交易中导入电子邮件 ID。

点击"操作" > "导入订阅者"，然后选择交易类型（如线索）。点击"导入"，所有在电子邮件字段中提到的电子邮件 ID 将被导入。

### 2.2 电子邮件模板

您可以为邮件组指定确认电子邮件和欢迎电子邮件模板。确认电子邮件将用于确认用户订阅您的新闻通讯。当用户确认订阅您的电子邮件新闻通讯时，将发送欢迎电子邮件。

确认电子邮件模板：此电子邮件模板用于替换新闻通讯订阅时发送的默认模板。当有人订阅邮件组网站时，通过此模板生成的确认电子邮件将被发送。

欢迎电子邮件模板：当订阅者在确认后被添加到邮件组时，这里设置的电子邮件模板将被发送给他们。
