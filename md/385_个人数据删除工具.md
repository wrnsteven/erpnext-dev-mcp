# 个人数据删除工具

个人数据删除工具使用户能够删除其账户，并匿名化用户在使用 ERPNext 期间生成的所有个人可识别数据。也就是说，个人可识别信息将被随机化。这包括您用户账户中的个人可识别数据，如：用户名、全名、出生日期、电话号码、手机号码、位置、兴趣、个人简介、邮件签名、电子邮件、联系人、地址、通信记录等。它还包括来自线索和商机中的数据，您保存的详细信息，如电话号码、手机号码、传真、网站和名称。

但是，这不包括法律要求企业保留的数据。

## 如何申请账户删除

要开始删除用户账户和个人可识别数据，您需要访问 URL 字段中的 [host-name]/request-for-account-deletion（例如 example.erpnext.com/request-for-account-deletion）。

![图片](../../images/385_image727709.png)

输入与您 ERPNext 账户关联的电子邮件。提交请求后，您将收到成功响应。

![图片](../../images/385_imagea32960.png)

这将向与用户关联的电子邮件地址发送一封包含数据删除验证链接的电子邮件。

![图片](../../images/385_imagee38974.png)

- 用户点击验证链接后，将显示一条确认消息。

## 个人数据删除的工作原理

删除数据的请求记录在 doctype "Personal Data Deletion Request" 中。

![图片](../../images/385_personal-data-deletion-request-doctype.png)

此 doctype 维护三个状态来完成用户数据删除过程。

### 待验证

此状态表示用户已通过 web form 请求数据删除。但是，此请求的验证仍待处理。从搜索栏中搜索 Personal Data Deletion Request。

![图片](../../images/385_pending-verification.png)

### 待批准

这表示用户已通过电子邮件验证了请求。这使得系统管理员可以使用"删除数据"选项。

![图片](../../images/385_pending-approval.png)

### 已删除

这表示系统管理员已点击"删除数据"按钮。这意味着用户的个人可识别数据已被匿名化。

![图片](../../images/385_deleted-user.png)

### 为个人数据删除请求设置 SLA

您还可以通过网站设置设置个人数据删除请求的 SLA。这将显示在 web form 描述中。

- 进入网站设置

- 滚动到名为账户删除设置的部分

- 在 **账户删除 SLA（天）** 字段中设置用户账户删除请求将被满足的天数。

- 如果您启用 **在我的账户页面显示账户删除链接**，则表单链接将对网站我的账户页面上的用户可见
