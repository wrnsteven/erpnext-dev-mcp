# 集成类别

ERPNext 提供了广泛的集成能力，允许企业无缝连接各种第三方应用程序和服务，以提高运营效率和数据流通。

这些集成涵盖电子商务平台、支付网关、运输提供商、Google 服务、电子商务提供商等。通过将 ERPNext 与这些系统集成，企业可以简化流程、减少手动数据输入、提高数据准确性，并更深入地了解其运营情况，这最终有助于做出明智的业务决策。

要将 ERPNext 与此处未列出的其他应用程序集成，您可以使用 Frappe 的 REST API。访问此链接了解更多信息：

[Frappe REST API](https://frappeframework.com/docs/user/en/guides/integration/rest_api)
