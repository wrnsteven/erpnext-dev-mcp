# Document Versioning

Document Versioning helps you track changes made to any Form/Document.

To enable Document Versioning:

1. Go to Customize Form (Customize > Customize Form)
2. Select the Form for which you need to enable versioning
3. Check **Track Changes**

Now to view the version history of a document:

1. Open the document
2. Click on **Menu** (3-dot menu) in the top right corner
3. Click on **Version History**

![](/files/version_history.png)

You will see the version history of the document. To see the changes made in a particular version, click on the version and it will highlight the changes made in that version.

![](/files/version_history_2.png)
