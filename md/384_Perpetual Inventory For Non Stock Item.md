## 问题

已在公司主数据中启用了永续库存，但在某些采购发票过账时仍过账到费用科目。

## 答案

根据永续库存规则，只有当物料为库存项目时，其价值才会在采购时被记入"在途库存"。在这种情况下，采购发票的GL分录如下：

| **账户** | **借方** | **贷方** |
|---------|---------|---------|
| 债权人 | | 100 |
| 已收未验库存 | 90 | |
| 税额 | 10 | |

永续库存不适用于非库存项目，费用在采购发票提交时立即入账。在这种情况下，分类账的分录如下：

| **账户** | **借方** | **贷方** |
|---------|---------|---------|
| 债权人 | | 100 |
| 销货成本 / 其他费用科目 | 90 | |
| 税额 | 10 | |
