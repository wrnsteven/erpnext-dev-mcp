# 打印格式

**打印格式是您在打印或发送电子邮件时生成的文档布局。**

此功能适用于 ERPNext 中的所有业务单据，如所有销售和采购单据、HR 文档等等。

在 ERPNext 中，打印格式分为三种类型，分别是标准打印格式、自定义打印格式和 HTML 打印格式。

## 标准打印格式

ERPNext 中的每个可打印文档类型都拥有自己的标准打印格式，该格式由 Frappe Framework 自动生成。标准打印格式中的字段位置取决于相应字段在文档中的位置。

![图片](../../images/462_customize-standard-print-format.png)

对标准打印格式进行的任何更改都需要通过自定义表单来完成。您也可以查看[在打印格式中添加字段](/erpnext/making-fields-visible-in-print-format)。

## 自定义打印格式

您还可以使用名为[打印格式构建器](/erpnext/print-format-builder)的工具来创建自己的自定义打印格式。此工具可以帮助您通过拖放字段来制作符合您偏好的简单自定义打印格式。

![图片](../../images/462_customize-print-format.gif)

为了创建自定义打印格式，ERPNext 提供了三种样式的多个预定义模板，分别是：Modern（现代）、Monochrome（单色）和 Classic（经典）。

要创建您自己的版本，请从以下位置打开现有模板：

> 构建 &gt; 视图 &gt; 打印格式

## 打印设置

要编辑/更新您的打印和 PDF 设置，请前往：

> 设置 &gt; 打印设置

![图片](../../images/462_print-settings.png)

## HTML 打印格式

要创建 HTML 打印格式，您需要具备 HTML、CSS 和 Python 的一些知识。以下是一个具有特定格式的打印格式设计示例。

![图片](../../images/462_customize-custom-print-format-1.png)

打印格式在服务器端使用 [Jinja 模板语言](https://jinja.palletsprojects.com/en/3.0.x/templates/) 提供。所有表单都可以访问包含正在格式化文档信息的 doc 对象。您还可以通过 frappe 模块访问公共工具。在创建基于 HTML 的打印格式时获取支持，您可以参考 [ERPNext 社区论坛](https://discuss.erpnext.com/)，或就您的问题发表新帖。

对于样式，[Bootstrap CSS Framework](http://getbootstrap.com/) 已提供，您可以充分发挥其全部类的作用。

#### 参考资料

- [Jinja 模板语言：参考文档](https://jinja.palletsprojects.com/en/3.0.x/templates/)

- [Bootstrap CSS Framework](http://getbootstrap.com/)

#### 示例

```html
<h3>{{ doc.select_print_heading or "Invoice" }}</h3>
<div class="row">
    <div class="col-md-3 text-right">Customer Name</div>
    <div class="col-md-9">{{ doc.customer_name }}</div>
</div>
<div class="row">
    <div class="col-md-3 text-right">Date</div>
    <div class="col-md-9">{{ doc.get_formatted("invoice_date") }}</div>
</div>

        {%- for row in doc.items -%}

        {%- endfor -%}
    <table class="table table-bordered">
    <tbody>
        <tr>
            <th>Sr</th>
            <th>Item Name</th>
            <th>Description</th>
            <th class="text-right">Qty</th>
            <th class="text-right">Rate</th>
            <th class="text-right">Amount</th>
        </tr><tr>
            <td style="width: 3%;">{{ row.idx }}</td>
            <td style="width: 20%;">
                {{ row.item_name }}
                {% if row.item_code != row.item_name -%}
                <br>Item Code: {{ row.item_code}}
                {%- endif %}
            </td>
            <td style="width: 37%;">
                <div style="border: 0px;">{{ row.description }}</div>
            </td>
            <td style="width: 10%; text-align: right;">
                {{ row.qty }} {{ row.uom or row.stock_uom }}
            </td>
            <td style="width: 15%; text-align: right;">
                {{ row.get_formatted("rate", doc) }}
            </td>
            <td style="width: 15%; text-align: right;">
                {{ row.get_formatted("amount", doc) }}
            </td>
        </tr></tbody>
</table>

```

## 注意事项

- 要获取格式化的日期和货币值，请使用 `doc.get_formatted("fieldname")`

- 对于可翻译的字符串，请使用 `{{ _("This string is translated") }}`
