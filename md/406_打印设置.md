# 打印设置

在打印设置中，您可以设置打印偏好，例如纸张尺寸、默认字号、输出格式（PDF 或 HTML）等。

由于 ERPNext 是一款基于浏览器的应用程序，实际的打印命令由您使用的浏览器执行。

要编辑打印设置，请前往：

> 首页 > 设置 > 打印设置

![图片](../../images/406_print-settings.png)

打印设置中提供了多种配置选项，下面我们来逐一了解。

## 1. PDF 设置

### 1.1 PDF 或 HTML

当您从 ERPNext 发送任何文档（如销售订单/发票）的邮件时，该文件会以 PDF 或 HTML 格式发送。默认情况下，文件以 PDF 格式发送。如果您希望以 HTML 格式发送文档，只需取消勾选"发送打印件为 PDF"选项。

### 1.2 在 PDF 中重复页眉和页脚

信头是一种模板，您可以在其中定义标准的页眉和页脚，并将其附加到文档的打印格式中。如果启用此属性，页眉和页脚将被添加到每一页。如果您不希望页眉和页脚在每页重复，只需禁用此设置。

### 1.3 PDF 页面尺寸

打印 PDF 页面的默认尺寸为 A4，您可以从[此处](https://doc.qt.io/archives/qt-4.8/qprinter.html#PaperSize-enum)定义的选项中选择任意尺寸，包括以毫米为单位的自定义尺寸。

## 2. 页面设置

### 2.1 带信头打印

启用此属性后，打印文档时将自动勾选信头选项。请注意，您需要将信头设置为默认，或在单据中选择一个，才能使其显示在打印视图中。

### 2.2 紧凑型物料打印

销售订单/发票等单据包含详细列出所购或所售物料的表格，其中有多个列，如物料名称、描述、单位、单价、金额等。如果物料表格中有很多列，打印格式看起来会略显杂乱。您可以通过启用紧凑型物料打印来改善表格的显示效果。

根据此设置，打印格式中只会有四列，分别是：描述、数量、单价和金额。

其他列的值（如名称、描述、图片、序列号等）将被合并到描述列中。

### 2.3 允许打印草稿

单据（主要是业务单据）有两个身份验证阶段：保存和提交。已保存的单据为第一稿，尚未提交到系统。因此，在草稿阶段的单据会被限制打印。但是，如果您希望允许用户也打印草稿阶段的单据，请启用此复选框。

### 2.4 始终为草稿单据添加"草稿"标题

启用此设置还会在打印格式中打印"草稿"字样，以此表明所分享的文档尚未完全通过身份验证。

### 2.5 允许在表格内分页

如果某个物料的描述占据了页面过多的空间，启用此设置后，该物料的详细信息将被拆分到下一页。因此，页面分隔符将插入在物料描述和其他详细信息之间，其余内容将被推到下一页。

### 2.6 允许打印已取消单据

已取消的业务单据是不会对报表产生任何影响的单据。如果您希望允许打印已取消的业务单据，请启用此设置。单据只有在提交后才能被取消。

### 2.7 打印零金额税额

在销售和采购业务单据中，您可以添加多项税费。默认情况下，在打印格式中只显示已计算金额的税费。如果您还希望打印未适用且金额为零的税费，请启用此设置。

## 3. 网络打印机 / 打印服务器

您可以通过填写打印服务器的 IP 地址和端口来启用打印服务器，然后选择默认打印机。

在启用此功能之前，您需要安装 `pycups` 库。

如果系统中尚未安装 `cups` 库，您可能需要先安装它。

对于 Debian 系列操作系统：

`sudo apt-get install libcups2-dev`

对于 Red Hat 系列操作系统：

`sudo yum install cups-libs`

之后，使用以下命令在虚拟环境中安装 `pycups`：

`./env/bin/pip install pycups`

此操作应在 `frappe-bench` 目录下执行。

## 4. 原始打印

您可以启用原始打印并打印到多种支持的热敏打印机。了解更多详情，请阅读[原始打印](/erpnext/raw-printing)。
