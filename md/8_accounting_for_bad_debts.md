# 坏账会计

坏账是一种由于客户无法支付而导致的费用类型，当发票反映出这种情况下，如果有一种真诚或合理的信念认为该款项无法收回，将其核销是一种良好的做法。

它被视为业务损失，原本可以是收入的现被视为费用并转到费用账户下的核销账户。

考虑一个例子，Frank 有一张 200 美元的未结发票，他已经停业了，无法结清。

要核销，请按照以下步骤操作：

1. 创建类型为**核销条目**的**日记账分录**。
2. 选择**核销**账户借记，**债务人**贷记。
3. 选择交易对手类型为**客户**，交易对手为 **Frank**。
4. 选择你要核销的**参考类型**和**参考发票**（展开债务人的行并向下滚动到参考部分）。

一旦余额被核销，该金额将被记为费用，Frank 的未结余额将开始显示为 0。

将来，如果 Frank 以后有资格结清欠款或在下一年，创建反向日记账分录，将 Frank 从核销账户中移除，并用收到的付款进行调整。
