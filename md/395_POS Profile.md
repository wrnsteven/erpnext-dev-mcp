# POS Profile

在 ERPNext 中，POS Profile 允许在一定范围内使用“已付款”销售发票的 Point of Sale（销售点）功能。

POS 包含高级功能，可满足不同的业务需求，如库存管理、财务、仓储等，所有功能都内置于 POS 软件中。

如果您从事零售业务，您希望您的销售点系统尽可能快速和高效。为此，您可以为用户创建一个 POS Profile。

要访问 POS Profile 列表，请进入：

*首页 > 销售 > POS > POS Profile*

## 如何创建 POS Profile

- 进入 POS Profile 列表，点击“新增 POS Profile”。

- 选择公司（Company）和仓库（Warehouse）。

- 添加付款方式。

- 设置货币、坏账冲销账户、坏账冲销成本中心和坏账冲销限额。

- 保存。

## 创建 POS Profile 时的附加选项

- **客户（Customer）：**如果在 POS Profile 上设置了客户，在销售发票/POS 发票上选择该 POS Profile 时，将自动设置相同的客户。

### 适用于用户

默认情况下，所有销售用户都可以访问 ERPNext 中创建的 POS Profile。但是，如果您只想让特定用户访问特定的 POS Profile，可以将他们添加到表格中。一旦在 POS Profile 中设置了至少一个用户，其他用户将无法使用该 POS Profile 进行零售交易。

设置默认 POS Profile：勾选表格中的“默认”复选框，当前 POS Profile 将成为该用户的默认 POS Profile。因此，下次该用户登录系统时，POS Profile 将被默认设置。

*注意：如果指定了特定用户，POS 设置将仅应用于该用户。如果用户选项留空，设置将对所有用户生效。了解 POS 的工作方式，请访问 Point of Sale 页面。*

### 付款方式

在表格中设置付款模式；如果此处未设置，默认将为现金。只有在此处设置的模式才会在使用 POS 时可用。添加付款模式后，通过勾选复选框将其中一种设置为默认付款方式。

要在退货发票上允许付款方式，请设置“允许退货”。

### 配置

POS Profile 中的配置部分仅适用于 ERPNext 中的销售点（POS）视图。这些设置控制 POS 界面的行为、布局和用户体验，不影响标准销售发票或其他视图。

- **隐藏图片**：默认情况下，POS 以网格视图显示商品。但是，启用“隐藏图片”配置将切换到列表视图。

- **忽略定价规则**：该 POS Profile 将忽略任何生效的定价规则。

- **隐藏无库存商品**：库存为零或负数的商品将不会显示。

- **允许用户编辑单价**：用户将可以在 POS 商品详情视图中编辑单价。

- **自动将筛选商品添加到购物车**：启用后，筛选的商品将自动添加到 POS 购物车，无需手动选择。仅适用于 POS 视图。

- **允许用户编辑折扣**：用户将可以在 POS 商品详情视图中编辑折扣。

- **保存时验证库存**：在保存发票时验证库存可用性。

- **将总计设置为默认付款方式**：在结账时自动将总计设置为默认付款方式。

- **订单完成时打印收据**：在订单完成时自动打开发票的打印预览。

- **允许部分付款**：允许对 POS 发票进行部分付款。

**新建发票时的操作**：在 POS 上点击新建发票按钮时，如果存在未保存的进行中订单发票，可执行以下操作。
- **始终询问**：询问用户是要保存进行中的发票还是放弃发票并加载新发票。

- **保存更改并加载新发票**：自动保存进行中的未完成发票并加载新发票。

- **放弃更改并加载新发票**：放弃未完成的发票并加载新发票。

### 设置商品组和客户组

在 POS Profile 中设置商品组/客户组后，在使用该 POS Profile 进行交易时，将自动选择该组。

### 打印设置

- **打印格式（Print Format）：**您可以设置一个打印格式，这将决定打印文档的布局样式。了解更多，请访问打印格式页面。

- **信头（Letterhead）：**您可以在公司的信头上打印 POS 销售发票。在此了解更多。

- **打印标题（Print Headings）：**POS 销售发票的打印标题也可以在打印文档时进行更改。例如，标题可以是“发票”或“账单”。您可以通过选择打印标题来实现此操作。创建新的打印标题请进入：首页 > 设置 > 打印 > 打印标题。在此了解更多。

- **条款和条件（Terms and Conditions）：**您销售的商品可能存在特定的条款和条件，可以在此处应用。了解如何添加条款和条件，请点击此处。

### 会计

- **价格列表（Price List）**：价格列表存储商品价格。在此设置价格列表将从该价格列表中获取当前 POS Profile 的商品价格。

- **货币（Currency）**：默认情况下，将根据公司的默认货币进行设置。但是，您可以更改它。如果您更改了货币，请记得同时更改相关账户。

- **税费和附加费（Taxes and Charges）**：在此选择销售税费和附加费模板或采购税费和附加费模板，将自动向 POS 交易应用相应的税费和附加费。

- **应用折扣于（Apply Discount On）**：您可以设置折扣是应用于总计（税前金额）还是净总计（税后金额）。

- **税种分类（Tax Category）**：在此选择税种分类后，与该税种分类关联的税务规则将应用于从该 POS Profile 执行的每笔交易。

可以设置以下账户，以便相应地更新总账：

- 找零账户
- 坏账冲销账户
- 坏账冲销成本中心
- 收入账户
- 费用账户

### 会计维度

会计维度允许您根据特定的区域、分支、客户等标记交易。这有助于根据所选标准分别查看会计报表。了解更多，请访问会计维度页面。

*注意：成本中心默认被视为一个维度。*
