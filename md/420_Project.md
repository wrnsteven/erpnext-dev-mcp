**项目是为查找某些信息、生产新事物或改进某些事物而设计的计划性工作。**

在 ERPNext 中，项目管理是以任务驱动的。您可以创建一个项目并将其划分为多个任务。

项目具有广泛的范围，因此可以细分为任务。例如，将"明年推出一款新智能手机"作为一个项目。那么设计、原型制作、测试、交付等工作就成为该项目的任务。

虽然项目中的每个任务都可以分配给个人或一组人员，但也可以在项目级别进行分配。

这些任务可以从项目本身创建，也可以单独创建任务。

要访问项目，请前往：

> 首页 > 项目 > 项目

![图片](../../images/420_imagec3fe96.png)

## 如何创建项目

- 进入项目列表并点击"新建"。

- 添加以下详细信息：

- **项目名称**：项目的标题。

- **状态**：项目的默认状态为"进行中"，以后可以更改为"已完成"或"已取消"。

- **预计结束日期**：输入您计划完成项目的日期。

### 创建项目时的其他选项

- **来自模板**：如果您有现有的[项目模板](/erpnext/project-template)，您可以选择使用此模板创建项目。

- **预计开始日期**：如果您已确定了项目的时间表，您可以在表单中定义预计开始日期和预计结束日期。

- **项目类型**：您可以将项目分类为不同的[类型](/erpnext/project-type)，例如内部或外部。

- **优先级**：您可以根据项目的重要程度选择项目的优先级级别。您也可以添加更多的优先级级别。

- **部门**：如果项目属于组织中的某个[部门](/hr/department)或由其拥有，您可以在此处添加。

- **是否激活**：是/否选项卡，允许您在以后的任何阶段更改项目的激活状态。

- **完成方式**：您可以根据以下三种方法之一来跟踪项目的完成百分比：**手动、任务完成、任务进度和任务权重**。

## 项目配置

### 客户、销售和访问控制

- **客户**：如果项目是为特定客户执行的，可以在此处输入详细信息。

**销售订单**：如果项目源自销售订单，您可以在这里获取其详细信息。这允许您：
- 跟踪项目执行与销售订单的对比情况

- 根据约定的范围向客户更新进度

**项目用户和访问权限**：您可以向项目添加网站用户以授予他们访问权限。这对于以下情况很有用：
- 希望监控进度或提供反馈的客户

- 参与项目的供应商、自由职业者或合同员工。您还可以：
- 添加用户时发送**欢迎邮件**

- 控制每个用户的**附件查看权限**

- **备注**：您可以向项目添加任何附加说明。

> 了解更多关于允许用户查看项目的信息 - 点击[此处](/erpnext/project-customer-portal)

![图片](../../images/420_imagebf5e91.png)

### **项目时间线跟踪**

- **实际开始日期**：实际开始日期和时间会根据链接到项目的第一条时间表自动获取。

- **实际结束日期**：实际结束日期和时间会根据项目的最新时间表更新自动记录。

> 了解更多关于时间表的信息，请点击[此处](/erpnext/timesheets)。

![图片](../../images/420_image1e72e3.png)

### 成本核算、开票和财务跟踪

- **预估成本**：输入项目的预估成本。

- **销售总额**：如果您已将项目与销售订单关联，销售订单总额将自动填充到这里。

- **成本总额**：系统将自动从链接到该项目的所有时间表中获取成本总额。

- **可开票总额**：系统将自动从链接到该项目的所有时间表中获取可开票总额。

- **费用报销总额**：基于员工为完成项目所申报的费用，费用报销总额将自动计算。

- **已开票总额**：已开票总额会使用针对销售订单创建的销售发票自动填充到系统中。

- **采购成本总额**：项目的采购成本总额是从为项目所需材料供应而发出的采购订单所生成的采购发票中获取的成本。

- **已消耗材料成本**：使用根据项目材料需求制作的库存调拨，捕获已消耗材料成本。

### 利润率

**毛利润**：毛利润为您提供成本总额与已开票总额之间的差额。

```
毛利润 = (销售总额 + 可开票总额) 
         - 成本总额 
         + 可开票总额 
         + 费用报销总额 
         + 采购成本总额 + 已消耗材料成本)
```

**毛利率 %**：已开票总额中用于成本总额的百分比构成毛利率。

```
 ((销售总额 + 可开票总额)
  - 成本总额 + 可开票总额
  + 费用报销总额 + 采购成本总额 
  + 已消耗材料成本) / 销售总额) * 100%
```

![图片](../../images/420_imagecf9da7.png)

### 监控进度

当您通过勾选复选框启用"收集进度"选项时，将允许您向项目添加监控详情。项目的进度报告将发送给项目的所有利益相关者。

- **假期列表**：您可以为您的公司选择假期列表。这将允许您仅在工作日收集进度报告。

- **频率**：您可以设置希望接收报告的频率。可以设置为每小时、每天两次、每天或每周。
